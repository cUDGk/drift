# DRIFT

完全ローカル完結型 AI チャットアプリ (Android)。
端末からインターネットに出ずに、キャラクターと会話するためのクライアント。

## これは何

クラウド LLM に投げたくない種類の会話 — ロールプレイ、個人的な相談、機微な話題 — を、端末の外へ一歩も出さずに行うためのチャットアプリ。

推論は端末内 (llama.cpp on NDK / arm64)。会話履歴もローカルストレージのみ。ネットワークは使わない。

## 目指していること

- **プライバシー 100% ローカル** — 会話が運営・サーバ・クラウドに送られない
- **擬似無限コンテキスト** — 小さいモデルでも長期会話が破綻しない記憶機構
- **キャラクターの一貫性** — 何ヶ月話しても相手が「忘れない」設計
- **ユーザーが記憶を握る** — 記憶はブラックボックスにせず、閲覧・編集・削除を可能に
- **モデル差し替え自由** — GGUF 形式で好きな量子化モデルを持ち込める
